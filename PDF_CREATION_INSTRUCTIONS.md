# How to Create PDF from HTML User Guide

## Quick Steps:

### Option 1: Using Your Browser (Recommended)

1. **Open the HTML file:**
   - Navigate to: `/Users/akash/Downloads/HM-ERP/HM_ERP_USER_GUIDE.html`
   - Double-click to open in your default browser
   - OR right-click → Open With → Chrome/Safari/Firefox

2. **Print to PDF:**
   - Press `Cmd + P` (Mac) or `Ctrl + P` (Windows)
   - In the print dialog:
     - **Destination:** Select "Save as PDF"
     - **Layout:** Portrait
     - **Paper Size:** A4
     - **Margins:** Default
     - **Background graphics:** ✓ Enabled (to include colors)
   
3. **Save:**
   - Click "Save" or "Print"
   - Name it: `HM_ERP_Complete_User_Guide.pdf`
   - Save to: `/Users/akash/Downloads/HM-ERP/`

### Option 2: Using Online Converter

1. Go to: https://www.sejda.com/html-to-pdf
2. Upload: `HM_ERP_USER_GUIDE.html`
3. Click "Convert to PDF"
4. Download the PDF

### Option 3: Using Mac Preview

1. Open `HM_ERP_USER_GUIDE.html` in Safari
2. File → Export as PDF
3. Save as `HM_ERP_Complete_User_Guide.pdf`
